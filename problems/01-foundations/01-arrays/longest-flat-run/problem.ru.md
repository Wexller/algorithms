# Longest Flat Run

## Summary

Find the length of the longest contiguous run of equal numbers.

## Input

`values: number[]`.

## Output

The maximum run length.

## Examples

### Example 1

- Input: values = [3, 3, 3, 1, 1, 5]
- Output: 3
- Explanation: The first run has length 3.

## Constraints

- Return 0 for an empty array.
