# First Unique Signal

## Summary

Return the index of the first character that appears exactly once in the string. Return `-1` if no such character exists.

## Input

`text: string`.

## Output

Zero-based index or `-1`.

## Examples

### Example 1

- Input: text = "swiss"
- Output: 1
- Explanation: The first unique character is `w`.

## Constraints

- Treat uppercase and lowercase as different characters.
