# Longest Consecutive Days

## Summary

Given unsorted day numbers, find the length of the longest consecutive run.

## Input

`days: number[]`.

## Output

Maximum consecutive run length.

## Examples

### Example 1

- Input: days = [100, 4, 200, 1, 3, 2]
- Output: 4
- Explanation: The run 1, 2, 3, 4 has length 4.

## Constraints

- Duplicate values may appear.
