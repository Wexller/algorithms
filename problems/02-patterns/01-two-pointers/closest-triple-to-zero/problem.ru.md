# Closest Triple To Zero

## Summary

Find a triple whose sum is closest to zero and return that sum.

## Input

`values: number[]`.

## Output

The closest sum.

## Examples

### Example 1

- Input: values = [-4, -1, 1, 2]
- Output: -1
- Explanation: The closest triple is -4 + 1 + 2 = -1. No triple sums exactly to zero.

## Constraints

- Assume at least three numbers.
