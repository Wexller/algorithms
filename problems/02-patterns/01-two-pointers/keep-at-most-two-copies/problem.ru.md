# Keep At Most Two Copies

## Summary

Given a sorted array, rewrite it in place so each value appears at most twice. Return the new logical length.

## Input

`values: number[]`.

## Output

New length after in-place compaction.

## Examples

### Example 1

- Input: values = [1, 1, 1, 2, 2, 3]
- Output: 5
- Explanation: The array prefix becomes `[1, 1, 2, 2, 3]`.

## Constraints

- The relative order of kept values must stay the same.
