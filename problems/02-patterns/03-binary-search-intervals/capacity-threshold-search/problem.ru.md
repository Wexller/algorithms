# Capacity Threshold Search

## Summary

Given package weights and a number of days, find the minimum daily capacity that can ship everything in time.

## Input

`weights: number[]`, `days: number`.

## Output

Minimum feasible capacity.

## Examples

### Example 1

- Input: weights = [1, 2, 3, 1, 1], days = 4
- Output: 3
- Explanation: Capacity 3 is the smallest value that works.

## Constraints

- Packages must stay in original order.
