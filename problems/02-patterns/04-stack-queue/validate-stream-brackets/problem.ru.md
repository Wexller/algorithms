# Validate Stream Brackets

## Summary

Return `true` if every opening bracket is closed in the correct order.

## Input

`stream: string`.

## Output

Boolean validity.

## Examples

### Example 1

- Input: stream = "{[()]}"
- Output: true
- Explanation: Every opener closes in the reverse order.

## Constraints

- Input consists only of `()[]{}`.
