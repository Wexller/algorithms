# Odd Even Reorder

## Summary

Reorder a linked list so nodes at odd indices come first, followed by nodes at even indices.

## Input

`head: ListNode | null`.

## Output

Reordered head.

## Examples

### Example 1

- Input: head = [1, 2, 3, 4, 5]
- Output: [1, 3, 5, 2, 4]
- Explanation: Odd-position nodes stay in order, then even-position nodes.

## Constraints

- Indices are 1-based positions in the list.
