# Remove Nth Node From End

## Summary

Remove the nth node from the end of a singly linked list and return the head of the updated list.

## Input

`head: ListNode | null`, `n: number`.

## Output

The updated head.

## Examples

### Example 1

- Input: head = [1, 2, 3, 4, 5], n = 2
- Output: [1, 2, 3, 5]
- Explanation: The node with value 4 is removed.

## Constraints

- Assume `n` is valid for the current list length.
