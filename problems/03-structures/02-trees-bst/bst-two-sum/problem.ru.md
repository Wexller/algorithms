# BST Two Sum

## Summary

Return `true` when a BST contains two values whose sum equals the target.

## Input

`root: TreeNode | null`, `target: number`.

## Output

Boolean answer.

## Examples

### Example 1

- Input: root = [5, 3, 6, 2, 4, null, 7], target = 9
- Output: true
- Explanation: 2 + 7 = 9.

## Constraints

- Values may be negative.
