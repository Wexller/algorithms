# Level With Maximum Sum

## Summary

Return the zero-based level index whose node values sum to the largest total.

## Input

`root: TreeNode | null`.

## Output

Best level index, or `-1` for an empty tree.

## Examples

### Example 1

- Input: root = [1, 7, 0, 7, -8]
- Output: 1
- Explanation: Level 1 has sum 7, which beats level 0 and level 2.

## Constraints

- Use breadth-first or depth-first traversal.
