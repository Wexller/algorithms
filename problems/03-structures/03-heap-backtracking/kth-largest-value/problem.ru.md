# Kth Largest Value

## Summary

Return the kth largest value in an unsorted array.

## Input

`values: number[]`, `k: number`.

## Output

The kth largest element.

## Examples

### Example 1

- Input: values = [3, 2, 1, 5, 6, 4], k = 2
- Output: 5
- Explanation: The sorted order descending is 6, 5, 4, 3, 2, 1.

## Constraints

- Assume `1 <= k <= values.length`.
