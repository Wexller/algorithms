# Meeting Room Load

## Summary

Given meeting intervals, return the minimum number of rooms required.

## Input

`meetings: Array<[number, number]>`.

## Output

Minimum room count.

## Examples

### Example 1

- Input: meetings = [[0, 30], [5, 10], [15, 20]]
- Output: 2
- Explanation: Two overlapping meetings need two rooms.

## Constraints

- Intervals use `start < end`.
