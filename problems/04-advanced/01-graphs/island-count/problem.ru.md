# Island Count

## Summary

Count islands in a binary grid using 4-directional adjacency only.

## Input

`grid: string[][]` containing `'1'` and `'0'`.

## Output

Number of islands.

## Examples

### Example 1

- Input: grid = [[1,1,0],[0,1,0],[1,0,1]]
- Output: 3
- Explanation: The diagonal cells do not connect.

## Constraints

- Cells connect only up, down, left, and right.
