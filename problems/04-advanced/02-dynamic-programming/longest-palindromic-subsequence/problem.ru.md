# Longest Palindromic Subsequence

## Summary

Return the length of the longest subsequence that forms a palindrome.

## Input

`text: string`.

## Output

Maximum subsequence length.

## Examples

### Example 1

- Input: text = "bbbab"
- Output: 4
- Explanation: One longest palindromic subsequence is `bbbb`.

## Constraints

- Subsequence characters do not need to be contiguous.
